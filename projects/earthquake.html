Earthquake Prediction Model

This model demonstrates exceptional performance with 100% precision, recall, F1-score, and AUC. These metrics may indicate an idealized synthetic dataset, potential overfitting, or feature leakage (e.g., lag features inadvertently revealing future events).

Preprocessing includes temporal feature extraction (hour, day, month), daily seismic metric aggregation, and lag features (1, 7, 14 days), though it currently lacks spatial analysis for location-specific prediction.

Interpretability is enhanced using visualizations like PCA projections, predicted vs. actual probabilities, and feature importance rankings (highlighting lagged occurrences, depth, and magnitude as key predictors). However, limitations include uncalibrated confidence scores, binary classification (no magnitude/location prediction), and potential data leakage.

Future improvements should integrate geospatial features, calibrate probabilities, and implement rigorous out-of-sample validation to ensure real-world reliability.

Summary:

  • Preprocessing: Extracted temporal features (hour/day/month), aggregated seismic metrics with lag features (1/7/14 days), spatial data missing
  • Model Insights: 100% evaluation metrics suggest idealized or overfit model, lacking generalization across real-world datasets
  • Interpretability: Visual tools (PCA, importance rankings) aid transparency; issues include data leakage and limited prediction scope
  • Enhancements Needed: Add geospatial features, confidence score calibration, and robust validation strategies
